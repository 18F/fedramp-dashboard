Class: GridSearch

Components.GridSearch

new GridSearch()

Allows a dataset to be searched using free text searches. This component utilizes a property expression to determine which field to search on. See Searcher example for property expression samples. If no property is specified, a custom filterFunc can be passed in to include custom filtering logic.
Source:

Example
<grid-search property="product.name in products" placeholder="Search by Product Name"></grid-search>

<grid-search filter-func="myCustomFilterFunc" placeholder="Search by Product Name or Provider Name"></grid-search>
